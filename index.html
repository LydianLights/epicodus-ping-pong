Ping Pong Simulation 20XX

How to play:

It's easy, just:

  1. Select a game!
  2. Type in a number!
  3. Press the button!
  4. ENJOY!!

Ping Pong

Rules:

Enter a number to count up to:

  • Numbers divisible by 3 will PING! How exciting!
  • Numbers divisible by 5 will PONG! How daring!
  • Numbers divisible by both 3 and 5 will PING-PONG! How amazing!

Prime Pong

Rules:

Enter a number to count up to:

  • All prime numbers will PRIME! How expected!
  • Every 3rd prime number will PRIME-PING! How bold!
  • Every 5th prime number will PRIME-PONG! How frightening!
  • Every number that is both an every-3rd-prime and an every-5th-prime will PRIME-PING-PONG! How mathematical!
